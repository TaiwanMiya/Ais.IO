以下是一些常用的 Linux 指令，按类别进行整理，方便参考和使用：

---

### 1. **文件和目录操作**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `ls`          | 列出当前目录的文件和子目录。               |
| `cd`          | 切换目录。                                |
| `pwd`         | 显示当前工作目录路径。                     |
| `mkdir`       | 创建一个新目录。                           |
| `rmdir`       | 删除空目录。                               |
| `rm`          | 删除文件或目录（使用 `-r` 删除目录及内容）。|
| `cp`          | 复制文件或目录。                           |
| `mv`          | 移动或重命名文件或目录。                   |
| `touch`       | 创建一个新的空文件。                       |
| `find`        | 查找文件或目录。                           |
| `tree`        | 树状结构显示目录内容（需安装）。            |

---

### 2. **文件内容查看**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `cat`         | 显示文件内容。                              |
| `more`        | 分页查看文件内容。                          |
| `less`        | 分页查看文件内容，支持上下滚动。             |
| `head`        | 查看文件的前几行。                          |
| `tail`        | 查看文件的最后几行，使用 `-f` 动态跟踪日志。 |
| `nano`        | 简单的文本编辑器。                          |
| `vim`         | 功能强大的文本编辑器。                      |

---

### 3. **文件权限**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `chmod`       | 修改文件权限。                              |
| `chown`       | 修改文件所有者。                            |
| `chgrp`       | 修改文件所属的组。                          |

---

### 4. **磁盘与存储**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `df`          | 显示磁盘空间使用情况。                      |
| `du`          | 显示目录或文件的大小。                      |
| `mount`       | 挂载文件系统。                              |
| `umount`      | 卸载文件系统。                              |
| `lsblk`       | 显示块设备信息。                            |
| `blkid`       | 显示设备的 UUID 和文件系统类型。             |
| `fdisk`       | 磁盘分区工具。                              |

---

### 5. **进程管理**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `ps`          | 显示当前运行的进程。                        |
| `top`         | 实时显示系统资源使用情况。                  |
| `htop`        | 更友好的实时进程监控工具（需安装）。        |
| `kill`        | 终止指定的进程。                            |
| `killall`     | 按名称终止所有匹配的进程。                  |
| `jobs`        | 显示后台运行的任务。                        |
| `bg`          | 将任务移到后台运行。                        |
| `fg`          | 将后台任务移到前台运行。                    |

---

### 6. **网络相关**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `ping`        | 测试网络连通性。                            |
| `curl`        | 发送 HTTP 请求。                            |
| `wget`        | 下载文件。                                  |
| `ifconfig`    | 查看或配置网络接口（需要安装 net-tools）。  |
| `ip`          | 显示或配置网络接口（推荐）。                |
| `netstat`     | 显示网络连接情况（需安装）。                |
| `ss`          | 查看网络连接状态（替代 netstat）。          |
| `scp`         | 远程复制文件。                              |
| `rsync`       | 高效同步文件和目录。                        |
| `ssh`         | 远程登录到另一台主机。                      |

---

### 7. **系统管理**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `uname`       | 显示系统信息。                              |
| `hostname`    | 显示或设置主机名。                          |
| `uptime`      | 显示系统运行时间及负载。                    |
| `who`         | 显示当前登录用户。                          |
| `whoami`      | 显示当前用户的名称。                        |
| `id`          | 显示用户的 UID 和 GID。                     |
| `env`         | 显示环境变量。                              |
| `date`        | 显示或设置系统时间。                        |
| `shutdown`    | 关闭或重启系统。                            |
| `reboot`      | 重启系统。                                  |

---

### 8. **压缩与归档**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `tar`         | 打包和解压文件。                            |
| `gzip`        | 压缩文件。                                  |
| `gunzip`      | 解压 `.gz` 文件。                           |
| `zip`         | 压缩文件为 `.zip` 格式。                    |
| `unzip`       | 解压 `.zip` 文件。                          |
| `xz`          | 压缩文件为 `.xz` 格式。                     |
| `unxz`        | 解压 `.xz` 文件。                           |

---

### 9. **用户管理**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `adduser`     | 添加用户。                                  |
| `passwd`      | 修改用户密码。                              |
| `su`          | 切换用户。                                  |
| `sudo`        | 使用超级用户权限执行命令。                  |
| `deluser`     | 删除用户。                                  |

---

### 10. **其他实用工具**
| 指令           | 作用                                         |
|----------------|--------------------------------------------|
| `alias`       | 创建命令别名。                              |
| `history`     | 查看命令历史记录。                          |
| `clear`       | 清屏。                                      |
| `echo`        | 输出文本或变量内容。                        |
| `tee`         | 输出到终端并保存到文件。                    |

掌握这些指令可以有效提高 Linux 下的工作效率。建议结合 `man` 查看每个命令的详细使用方法，例如 `man ls`。
